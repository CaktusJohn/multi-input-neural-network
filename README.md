# Мультивходовая Нейронная Сеть для Прогнозирования Возраста по когнитивным Тестам

## О проекте
Модель предсказывает возраст по результатам пяти когнитивных тестов. Используется многоветочная архитектура на PyTorch: каждый тест поступает в свою входную ветку, обрабатывается LSTM, после чего признаки объединяются и подаются в финальный регрессионный блок.

## Архитектура
- пять входных веток (по одной на тест);
- LSTM в каждой ветке для извлечения временных признаков;
- конкатенация всех признаков в общий вектор;
- полносвязная «голова» для регрессии возраста;
- Dropout для регуляризации.

## Файлы
### `prepare_multi_test_data.py`
Подготовка данных: группировка по участникам, обработка пропусков, паддинг, сохранение `.npy`.

### `multi_input_model.py`
Определение многоветочной архитектуры (LSTM + объединение + регрессия).

### `train_multi_input.py`
Обучение модели: загрузка данных, оптимизация, сохранение результатов.

### `notebooks/`
Jupyter Notebook для анализа и визуализации.
