# Мультивходовая нейронная сеть для прогнозирования возраста по когнитивным тестам

## О проекте
Модель предсказывает возраст по результатам пяти когнитивных тестов. Используется многоветочная архитектура на PyTorch: каждый тест поступает в свою входную ветку, обрабатывается LSTM, после чего признаки объединяются и подаются в финальный регрессионный блок. Также реализована модель с одним входом для всех тестов.

## Архитектура
- пять входных веток (по одной на тест)
- LSTM в каждой ветке для извлечения временных признаков
- конкатенация всех признаков в общий вектор
- полносвязная «голова» для регрессии возраста
- Dropout для регуляризации

## Описание файлов

- **prepare_multi_test_data.py** — подготовка и обработка данных для мультивходовой модели.
- **multi_input_model.py** — архитектура мультивходовой нейронной сети (LSTM-ветки + регрессия).
- **train_multi_input_model.py** — обучение мультивходовой модели.
- **nerual_network.ipynb** — обучение и анализ одновходовой нейронной сети (RNN/LSTM).
- **model_training_and_evaluation.ipynb** — обучение и сравнение мультивходовой модели, анализ метрик.
- **data_prapation.ipynb** — подготовка и исследование данных.
